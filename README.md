# Битовые операции и шахматные фигуры

Учебный проект по реализации битовых алгоритмов и шахматных фигур на C++.

## Структура проекта

### **Исходный код (src/)**:
- [`Bits.cpp`](src/Bits.cpp) - основной класс с реализацией алгоритмов
- [`test.cpp`](src/test.cpp) - модульные тесты (TestCounts и TestBitboard)
- [`main.cpp`](src/main.cpp) - точка входа с демонстрацией работы

### **Отчеты (report/)**:
- [`report.md`](report/report.md) - полный отчет по выполнению задания
- [`output.txt`](report/output.txt) - консольный вывод тестирования

## Реализованные алгоритмы

### **Подсчет битов**:
- Наивная реализация
- Алгоритм через вычитание единицы
- Кэшированная версия (с оптимизацией)

### **Шахматные фигуры**:
- Король, Конь, Ладья, Слон, Ферзь
- Битовое представление ходов
- Оптимизация через магические константы

## Особенности
- Проект собран на CMake
- Сравнение производительности алгоритмов
- Битовые оптимизации для шахматных фигур
